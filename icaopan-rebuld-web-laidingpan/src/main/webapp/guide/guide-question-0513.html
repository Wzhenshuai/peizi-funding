1)我的资金安全吗？
您的每一笔保证金的流转在银行均有记录，i操盘与您签署的协议符合相关法律规定。您的股票交易受券商监管，每一笔交易都可以在level2行情查到。
2)收不到验证码短信怎么办？
  1. 1.确认手机号码是否填写正确；
  2. 2.请确认手机是否安装短信拦截或过滤软件；
  3. 3.请确认手机是否能够正常接收短信（信号问题、欠费、停机等）；
  4. 4.短信收发过程中可能会存在延迟，请耐心等待，短信在5分钟内均有效；
  5. 5.请联系在线客服或致电4001-667-067。
3)为什么点击充值，第三方支付页面没有出现？
点击充值后会跳转到第三方支付充值页面，请检查浏览器是否阻止弹出窗口，如有，请选择允许i操盘网站的所有弹窗操作。
4)为什么显示充值失败？
  1. 1.充值金额超过银行限额。请注意第三方支付充值页面下方所选择银行的支付限额，不同银行的限额可能不同；
  2. 2.如没有超过银行限额，有可能是您使用的网络延迟导致，建议您稍后再试；
  3. 3.充值过程中如有问题请联系在线客服或致电4001-667-067。
5)我在充值的时候提示“异常”或者“充值失败”，但是钱已经扣了怎么办？
请您登录i操盘账户，查询现金账户资产与充值明细，如确实与您的充值金额不符，请截图并发送给在线客服,并提供您的注册手机号、姓名和身份证号码。我们会查询后尽快处理并给您反馈。
6)如何提取盈利？
请先将股票账户可用余额转出到现金账户，再绑定实名银行卡后提交提现申请即可
申请盈利转出，必须满足【股票总资产>总操盘资金】时才可以操作
  1. A：股票账户可用余额>股票总资产−初始总操盘资金
    盈利可转出金额 =股票总资产−初始总操盘资金
    例如：股票总资产32000，初始总操盘资金30000，股票账户可用余额3000，盈利可转出金额就为2000；
  2. B：股票账户可用余额 < 股票总资产−初始总操盘资金
    盈利可转出金额 =股票账户可用余额。
7)提现时提示“取现失败”或“异常”怎么办？
请您登录i操盘账户，查询现金账户资产与提现明细，如确实提现失败，请截图并发送给在线客服，并提供您的注册手机号、姓名、身份证号码。我们会及时处理并给您反馈。
8)为什么我的提现未到账？
  1. 1. 银行开户行信息填写错误；
  2. 2. 银行账号/户名填写错误，或是账号与户名不符；
  3. 3. 银行账户冻结或正在办理挂失；
  5. 如果遇到以上情况，我们会在收到银行转账失败的通知后解除您的提现资金冻结，并及时通知您相关信息，请您不必担心资金安全。如有问题请联系在线客服或致电4001-667-067。
9)设定实名后，还可以修改吗？
根据国家金融监管机构要求，为保障您的账户资金安全，需要您进行实名认证。实名认证后认证信息不可修改。
10）配资炒股需要支付哪些费用？
在i操盘配资，平台只收取全网最低的利息，不收取任何其他费用。股票交易还需缴纳交易手续费，包含印花税、沪市股票过户费和交易佣金，印花税和过户费按财政部和交易所规定收取，交易佣金0.08%（万分之八）由券商收取。炒股赚钱之后，所有的盈利100%都归您。
11）配资审核需要多久？
交易日工作时间（9:00-18:00）申请，1小时内即可完成审核；非交易日申请，审核会在下一个交易日上午9:30前完成审核。
12）配资后何时开始计算利息？
从平台配资审核通过起开始计算。
13）多个配资项目是否可以使用一个股票交易账户？
可以。平台支持多次配资，一个配资项目到期时只需将此项目的配资金额归还平台，剩余资金可以继续用于股票操盘，最后一个配资项目到期时，您的保证金及盈利将会全部转出至现金账户中。
14)我可以提前还款吗？
  1. 您可以选择在配资到期前的任何一天提前还款，在“我的账户——实盘项目”的配资项目中选择相应的项目“提前还款”即可，提前还款需缴纳违约金：
  2. 当配资周期剩余0个月内时，提前还款无需缴纳违约金；
  3. 当配资周期剩余2个月内时，提前还款需缴纳下个月利息20%的违约金；
  4. 当配资周期剩余3-6个月时，提前还款需缴纳下个月利息40%的违约金。
